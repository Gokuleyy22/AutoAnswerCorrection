Q9. What is a linked list? Explain its types with advantages. [10 marks]

Your Answer: A linked list is a data structure where each element is stored in a node. Each node has data and a pointer to the next node. There are three types: singly linked list where nodes point to next node, doubly linked list where nodes have both next and previous pointers, and circular linked list where last node points to first.
